# 🤖 Умный ТГ Ассистент + LM Studio

Персональный ИИ-ассистент в Telegram. Пишешь ему — он думает через LM Studio — отвечает. 
Может отправлять сообщения другим пользователям по шаблонам с кодовыми словами.

---

## 📦 Структура проекта

```
tg_assistant/
├── main.py                  ← Точка входа
├── requirements.txt
├── .env.example             ← Шаблон конфига (скопируй в .env)
├── config/
│   ├── settings.py          ← Загрузка настроек из .env
│   └── templates.json       ← Твои шаблоны сообщений (создаётся автоматически)
├── core/
│   ├── lm_client.py         ← Клиент LM Studio API
│   ├── template_manager.py  ← Система шаблонов с кодовыми словами
│   ├── message_handler.py   ← Роутер + логика обработки команд
│   └── sender.py            ← Отправка сообщений другим юзерам
├── sessions/                ← Telegram сессии (создаётся автоматически)
└── logs/                    ← Логи (создаётся автоматически)
```

## 🚀 Быстрый старт

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

### 2. Получи Telegram API ключи

Зайди на [my.telegram.org/apps](https://my.telegram.org/apps) → создай приложение.  
Тебе нужны **`api_id`** и **`api_hash`**.

> ⚠️ **Используй ОТДЕЛЬНЫЙ Telegram аккаунт для ассистента!**  
> Не используй свой основной — userbot может быть заблокирован если злоупотреблять.

### 3. Узнай свой Telegram ID

Напиши боту [@userinfobot](https://t.me/userinfobot) → он пришлёт твой числовой ID.

### 4. Настрой конфиг

```bash
cp .env.example .env
# Открой .env и заполни все поля
```

Минимальный `.env`:
```env
TG_API_ID=12345678
TG_API_HASH=abcdef1234567890abcdef1234567890
TG_PHONE=+79991234567
TG_OWNER_ID=123456789
```

### 5. Настрой LM Studio

1. Открой LM Studio
2. Загрузи любую модель (например `Mistral 7B`, `Llama 3.1 8B`, `Qwen 2.5`)
3. Перейди в **Local Server** (иконка `<>`)
4. Нажми **Start Server**
5. Убедись что сервер запущен на `http://localhost:1234`

### 6. Запусти ассистента

```bash
python main.py
```

При первом запуске Telegram попросит код подтверждения — введи его.  
Сессия сохранится в `sessions/` — следующие запуски без кода.

## 💬 Использование

### Обычный вопрос
Просто напиши ассистенту — он ответит через нейронку:
```
Какая погода будет завтра в Москве?
Переведи на английский: добрый вечер
Напиши функцию на Python для сортировки списка
```

### Отправка по шаблону

**Формат:**
```
[кому] отправить [КОДОВОЕ_СЛОВО] [переменная=значение]
```

**Примеры:**
```
Лимона отправить ПЕРЕДАЧА_ДОКУМЕНТОВ тема=договор аренды
@username ВСТРЕЧА когда=завтра в 14:00 место=офис тема=новый проект
Написать @test123 СПАСИБО за_что=помощь с переездом
```

### Системные команды
```
/шаблоны          — показать все шаблоны
/история          — последние сообщения диалога
/очистить         — очистить историю диалога
/перезагрузить шаблоны  — обновить шаблоны без рестарта
/помощь           — справка
```

## 📋 Настройка шаблонов

Шаблоны хранятся в `config/templates.json`. При первом запуске создаётся пример.

```json
{
  "templates": [
    {
      "keyword": "МОЙ_ШАБЛОН",
      "name": "Название шаблона",
      "description": "Описание когда использовать",
      "text": "Привет! Пишу тебе по поводу {тема}.\n\nДетали: {детали}\n\nЖду ответа."
    }
  ]
}
```

- **`keyword`** — кодовое слово (регистр не важен)
- **`text`** — текст с переменными в `{фигурных_скобках}`
- Переменные передаются в команде через `=`: `тема=договор детали=подробности`

После изменения файла напиши ассистенту: `/перезагрузить шаблоны`

## ⚙️ Продвинутые настройки (.env)

```env
# Системный промпт — как ведёт себя нейронка
SYSTEM_PROMPT=Ты мой личный ассистент Лимон. Отвечай кратко, без воды.

# Параметры модели
LM_MAX_TOKENS=2048        # длиннее ответы
LM_TEMPERATURE=0.3        # менее творческий, более точный
LM_TIMEOUT=180            # больше времени для думалки

# Конкретная модель (если в LM Studio несколько)
LM_MODEL=lmstudio-community/Meta-Llama-3.1-8B-Instruct-GGUF
```

## ❓ Возможные проблемы

| Проблема | Решение |
|----------|---------|
| `LM Studio недоступен` | Запусти LM Studio и включи Local Server |
| `FloodWaitError` | Подожди указанное время, ты отправляешь слишком быстро |
| `UserPrivacyRestrictedError` | Пользователь запретил сообщения от незнакомцев |
| `UsernameNotOccupiedError` | Username не существует, проверь написание |
| Ассистент не отвечает | Проверь `TG_OWNER_ID` — должен совпадать с твоим ID |
